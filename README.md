# Python Sales & Inventory Management System

A console-based sales and inventory management application developed in Python for managing product stock, processing sales transactions, generating invoices, and maintaining inventory records.

## Features

- Inventory Management
- Product Stock Tracking
- Sales Processing
- Invoice Generation
- VAT Bill Generation
- File-Based Data Storage
- Automated Discount Calculation
- Real-Time Stock Updates

## Technologies Used

- Python
- File Handling
- Lists
- Dictionaries

## Project Structure

- main.py
- operation.py
- read.py
- write.py
- products.txt

## Key Functionalities

### Sales Processing
- Process customer purchases
- Generate invoices automatically
- Update inventory after each transaction

### Inventory Management
- View available products
- Update stock levels
- Track product information

### Discount System
- Buy 3 Get 1 Free promotion
- Automatic discount calculations
